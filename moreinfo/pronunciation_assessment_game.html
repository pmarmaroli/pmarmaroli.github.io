User Interface and Experience

  • Sentence Submission: Users can either write their own sentences or upload files following a specific template to begin the game.
  • Recording and Feedback: Users record their speech, and the system evaluates pronunciation accuracy, fluency, and completeness, providing real-time feedback.
  • Multilingual Support: The game supports multiple languages such as English, French, and Spanish, making it accessible to a diverse audience.
  • Interactive Progress: The game includes progress tracking through a series of milestones, motivating users to improve their pronunciation.

Frontend Development

  • Technologies Used: The frontend is developed using HTML, CSS, and JavaScript for a responsive and engaging user experience.
  • WaveSurfer.js Integration: WaveSurfer.js is used to display the waveform of recorded audio, allowing users to visualize their speech.
  • Dynamic Content: The game dynamically updates sentence feedback and color-codes words based on pronunciation accuracy.

Backend Development

  • Audio Processing: Node.js and Express handle the audio file uploads, processing, and communication with the Microsoft Speech SDK for pronunciation assessment.
  • Pronunciation Scoring: The backend uses the Microsoft Speech SDK to evaluate the speech against a reference sentence, providing detailed scores for accuracy, fluency, and prosody.
  • Database Management: Used session storage and database operations to handle user progress and feedback, ensuring smooth gameplay.

Post-Processing and Error Handling

  • Error Handling: The system implements error handling for audio processing, file uploads, and the Speech SDK to ensure a smooth user experience.
  • Audio Clean-Up: The system deletes temporary audio files after processing to optimize performance and storage.

Technical Stack and Tools

  • Frontend: HTML, CSS, JavaScript, WaveSurfer.js
  • Backend: Node.js, Express, Microsoft Speech SDK
  • Deployment: The project is deployed using cloud-based services and incorporates Azure for handling speech analysis.
