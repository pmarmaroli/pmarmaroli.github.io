Acoustic Vehicle Detection & Classification

Advanced MATLAB-based system using microphone arrays and deep neural networks for real-time vehicle identification

This project implements a sophisticated MATLAB-based acoustic detection system that identifies and classifies passing vehicles (cars, trucks, motorbikes) using advanced signal processing techniques. The system combines microphone array processing with custom-trained deep neural networks to provide real-time audio analysis and classification with visual feedback.

Technical Implementation

Signal Processing Architecture

  • Audio Capture: Real-time audio input via MATLAB's audioDeviceReader for live processing
  • File Processing: Support for pre-recorded audio files using audioread function
  • Cross-correlation Analysis: Multi-microphone signal evolution tracking for vehicle detection
  • Spectrogram Generation: Time-frequency domain conversion for CNN input preparation

Machine Learning Components

  • Detection CNN: Custom-trained network for vehicle presence detection using microphone arrays
  • Classification CNN: Single-microphone based neural network for vehicle type identification
  • Training Dataset: Internally recorded and manually labeled vehicle audio data
  • Model Optimization: MATLAB deep learning toolbox optimization for real-time performance

Visualization & Interface

  • Real-time Plotting: Dynamic probability displays for each vehicle class
  • Data Buffering: Temporal smoothing of classification results for stability
  • Webcam Integration: Screenshot capture during detection events for documentation
  • Error Handling: Robust audio format validation and sampling rate correction
