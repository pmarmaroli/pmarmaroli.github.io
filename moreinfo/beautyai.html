About BeautyAI

The development of BeautyAI started with the goal of creating an accessible platform for assessing skin quality using modern AI technologies. The project leverages AI models to provide detailed skin quality assessments, making them more accessible to users globally.

Overview

BeautyAI is an AI-powered platform designed to facilitate the analysis of skin quality based on user-provided images. The development process focused on integrating advanced AI models to ensure accurate assessments and provide personalized skincare recommendations.

User Interface and Experience

  • Camera Integration: Users can capture images directly through the platform, ensuring real-time analysis and feedback.
  • Assessment Display: Detailed assessment results are displayed with information about various skin issues and recommended solutions.

Backend Functionality

  • Image Processing: Utilized Sharp for image resizing to ensure optimal dimensions for analysis. Images are converted to Base64 encoding for AI model processing.
  • AI Integration: Integrated OpenAI's GPT-4 model to analyze skin images and generate detailed reports, including the severity of skin issues and recommended natural solutions.
  • Data Management: Implemented file system operations to store intermediate data like Base64 encoded images and JSON results. A cron job was set up to clear temporary data periodically.

Stripe Payment Integration

  • Payment Handling: Configured Stripe to handle payment sessions, enabling users to unlock detailed reports upon successful payment.
  • Payment State Management: Managed payment states to blur/unblur content based on payment status, enhancing the user experience.

Post-Processing and Error Handling

  • Error Handling: Implemented robust error handling throughout the image processing and AI integration steps to ensure smooth user experience.

Technical Stack and Tools

  • Frontend Development: The frontend is developed using HTML, CSS, and JavaScript, providing a responsive and user-friendly interface.
  • Backend Development: The backend uses Node.js and Express for handling API requests, data processing, and integration with AI models.
  • Version Control and Deployment: Git is used for version control, and the application is set up for deployment on Azure using GitHub Actions for CI/CD.
