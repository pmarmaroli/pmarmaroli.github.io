About TroubleShot

TroubleShot aims to simplify the DIY troubleshooting process by using image analysis to provide step-by-step instructions and a list of necessary tools. Users can upload images of their problems and receive tailored solutions.

User Interface and Experience

  • Image Capture: Users can capture and upload images directly through the platform for analysis.
  • Multilingual Support: The platform supports multiple languages including English, French, and Estonian, enhancing accessibility for a global audience.
  • Interactive Elements: The interface includes interactive elements like tool selection and instruction lists, making it easy for users to follow the provided solutions.

Frontend Development

  • Technologies Used: The frontend is built using HTML, CSS, and JavaScript, ensuring a responsive and user-friendly interface.
  • Language Switching: Implemented i18next for dynamic content translation based on the user's selected language.
  • Camera Integration: Used the MediaDevices API to enable users to capture images directly within the application.

Backend Development

  • Image Processing: Utilized Node.js and Express for handling image uploads and processing. Integrated OpenAI's GPT-4 model to analyze images and generate detailed reports.
  • Data Management: Implemented file system operations to store and manage uploaded images and analysis results.
  • Payment Integration: Configured Stripe to handle payments, allowing users to unlock detailed reports upon successful payment.

Post-Processing and Error Handling

  • Error Handling: Implemented robust error handling throughout the image processing and AI integration steps to ensure a smooth user experience.
  • Data Cleanup: Set up a cron job to periodically clear temporary data, maintaining optimal performance and storage management.

Technical Stack and Tools

  • Frontend: HTML, CSS, JavaScript, i18next
  • Backend: Node.js, Express, OpenAI API, Stripe API
  • Deployment: The application is deployed using a cloud-based platform, with GitHub Actions for CI/CD.
