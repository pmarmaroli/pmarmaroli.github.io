About HadithGPT

The development of HadithGPT started in spring 2024 with the goal of creating an accessible platform for searching and reading Hadiths using modern LLM technologies. The project leverages language models to provide meaningful translations and interpretations of Hadiths, making them more accessible to a global audience.

Overview

HadithGPT is an AI-powered platform designed to facilitate the search and retrieval of Hadiths based on user queries. The development process focused on integrating advanced language models to ensure accurate translations and efficient search capabilities.

User Interface and Experience

  • Language Support: The platform supports both English and Arabic. The interface dynamically adjusts based on the selected language.
  • Search Functionality: Users can input their queries through a text field, and the platform will retrieve the most relevant Hadiths based on the input. The search results are displayed with detailed information about each Hadith.

Backend Functionality

  • Translation Integration: The platform uses translation APIs to convert Arabic queries to English, ensuring broader accessibility. The translations are then processed by the language models to find relevant Hadiths.
  • Hadith Information Extraction: The backend processes Hadith data to extract relevant information such as narrator, book title, author, and Hadith number, which are displayed in the search results.
  • Data Management: The system efficiently manages large datasets of Hadiths, ensuring quick retrieval and accurate results.

Post-Processing and Error Handling

  • JSON Output Correction: After receiving search results, a post-processing step ensures that the output is correctly formatted and complete, handling any errors in the data.

Enhancing User Engagement

  • Interactive Elements: The search results include interactive elements such as links to source texts and additional information, enhancing the user experience.

Technical Stack and Tools

  • Frontend Development: The frontend is developed using HTML, CSS, and JavaScript, providing a responsive and user-friendly interface.
  • Backend Development: The backend uses Python and libraries such as Flask for handling API requests, data processing, and integration with AI models. Additional tools include PyTorch for model handling and JSON for data management.
  • Version Control and Deployment: Git is used for version control.
