About Cheffantastique

The development of Cheffantastique.com began in spring 2023, shortly after the public release of ChatGPT 3.5. Like many others, I was astonished by the capabilities of generative AI to produce meaningful responses to various prompts. One aspect that quickly became evident was the sensitivity of the AI to the phrasing of prompts. Even minor changes could result in significantly different outcomes. This observation inspired the creation of an interface that could guide users in crafting effective prompts. The theme of cooking was chosen somewhat randomly. Although I am not a chef or particularly passionate about food, this context served well as a proof of concept for the platform.

Overview

Cheffantastique.com is an AI-driven platform designed to generate custom recipes tailored to user preferences. The development process involved several key components, each aimed at creating a seamless and engaging user experience.

User Interface and Experience

  • Guided Interaction: The interface is designed to guide visitors through a series of questions. These questions help gather specific information about user preferences, dietary restrictions, and available ingredients. Users also have the option to let the bot generate a random recipe.
  • User Input Methods: Users can input their preferences through text fields, checkboxes, and dropdown menus, making the interaction intuitive and user-friendly.

Backend Functionality

  • Prompt Generation: The answers provided by users are formatted into an optimal prompt. This prompt is the payload for the ChatGPT API, which is responsible for generating the recipe.
  • ChatGPT API Integration: The platform integrates with the ChatGPT API to fetch custom recipes. The API returns the recipe in a structured JSON format, which includes details such as ingredients, quantities, step-by-step instructions, and cooking tips.
  • Data Security: The backend ensures the security and privacy of user data by following best practices in data encryption and storage.

Post-Processing and Error Handling

  • JSON Output Correction: After receiving the recipe from the ChatGPT API, a post-processing step is performed to correct potential errors in the JSON output. This ensures the recipe is properly displayed. Specific types of errors handled include formatting issues and missing data fields.

Enhancing User Engagement

  • Image Generation: To create a more engaging user experience, an image is generated based on the title of the recipe. This is achieved by integrating the DALL-E API, which provides a visual representation of the final dish.
  • Visual Appeal: The generated images are displayed alongside the recipe to make the presentation more appealing and to give users a better idea of the expected outcome.

Technical Stack and Tools

  • Frontend Development: The frontend is developed using Wix, incorporating HTML, CSS, and JavaScript for a responsive and interactive user interface.
  • Backend Development: The backend leverages Wix-Velo (JavaScript) to handle API requests, data processing, and integration with external services like ChatGPT and DALL-E APIs. Additional libraries and tools used include Wix-Data for database operations and Wix-Secrets for secure storage of API keys.
  • Version Control and Deployment: The development process uses Git for version control, ensuring efficient collaboration and management of code changes.
